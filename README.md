# Restaurant Website

This repository contains the source code for a restaurant website.

## Description
A modern and responsive restaurant website built using HTML, CSS, and JavaScript.
It is designed to showcase menu items, restaurant details, and contact information.

## Usage
This website is intended for commercial use by the purchasing restaurant.
The buyer is free to modify, customize, and deploy the website as required.
